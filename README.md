# 🌦️ Weather App

간단한 날씨 정보 확인 웹앱입니다.  
OpenWeather API를 활용하여 현재 날씨 정보를 확인할 수 있습니다.

## 📌 주요 기능
- 지역별 현재 날씨 조회
- 온도, 날씨 설명, 아이콘 표시
- 간단한 UI
